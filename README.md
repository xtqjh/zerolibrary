# Zerolibrary

## 创建库
`ng generate library share-desktop`

## 编译发布的包
`ng build share-desktop` 
`ng build share-mobile` 
`ng build share-jnc` 

## 上传发布的包
`cd dist/share-jnc`
`npm publish` 

## 删除发布的包
`npm unpublish ng-share-desktop@0.0.0-beta.0.0 --force`
`npm unpublish ng-share-mobile@0.0.0-beta.0 --force`
